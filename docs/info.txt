Why <i> was historically used
-----------------------------
Font Awesome originally used <i> because:
“i” stood for “icon”, not “italic” — a clever repurposing from early web days.
When Font Awesome launched (around 2012), <i> tags were lightweight, common for icons, and didn’t have much semantic meaning in that context.
It helped distinguish icon fonts from text elements in HTML, back when people used icon fonts instead of inline SVGs.
